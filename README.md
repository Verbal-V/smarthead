Система обнаружения проникновения
Это проект, который отслеживает людей на видео и определяет, заходят ли они в запрещённые зоны. Зоны можно разметить самому, система включает тревогу при нарушении.

Умеет:
Определять людей на видео (YOLO или простой детектор)
Показывать и проверять запрещённые зоны
Включать тревогу, если человек зашёл в зону
Автоматически отключать тревогу через 3 секунды
Сохранять зоны в JSON
Есть редактор зон

Как запустить:
pip install -r requirements.txt
Кидайте любое видео в data/src и переиминуйте его в your_video.mp4 
После этого
python zone_editor.py data/videos/your_video.mp4  
Редактируете видео, сохраняете и после
python run_system.py 
оно автоматом находит видео ваши выбираете его и запускаете
Либо
python main.py и путь к видео

Есть файл config.py
Там можете менять пороги обнаружения и т.д

Управление для zone_editor.py:
Нажимаете 1 - режим редактирования
Потом с ЛКМ ставите точки допустим 4 точки, нажимаете после этого ПКМ и S на англ клавиатуре что бы сохранить
d — удалить зону
q — выход

Зоны сохраняются в restricted_zones.json.

Как работает:
YOLO или простой детектор ищут людей
Центр человека проверяется, попал ли он в полигон
Если да — начинается тревога
Через 3 секунды после выхода — выключается


main.py — запуск с GUI
zone_editor.py — редактор зон
restricted_zones.json — сохранённые зоны
